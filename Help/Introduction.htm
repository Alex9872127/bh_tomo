Introduction

bh_tomo is an open sauce Matlab package developed to process and interpret borehole georadar data and ray-based 2D or 3D tomography. Bh_tomo runs under Matlab version 7.0 and above, and is therefore portable between computer operating systems supported by Matlab.

The main features of the package are:

  • Ray-based approach:

    - Travel time tomography

    - Attenuation tomography based on first-cycle amplitude

  • Two ray-tracing routines:

     - Straight rays in homogeneous velocity models

    - Curved rays in heterogeneous models

  • Two-dimensional domain:

    - Arbitrary position of transmitters & receiver in the 2D plane (arbitrary well trajectory)

  • Various data input format:

    - RAMAC (Mala Geosciences)

    - PulseEKKO (Sensors & Software)

    - SEGY

  • Handle of time units and tomography on seismic data as well

  • Two core inversion options:

    - Classic least-square solver

    - Geostatistical tomography and simulation

  • Elliptical anisotropy (with geostatistical approach)

  • Treatment of time-lapse data

  • Basic data base functionality with graphic user interface

  • 2D and 3D visualisations

  • Basic interpretation module for hydrogeology applications

  • Released under the GNU General Public License, version 2

The software package relies on a mini database and comprises interactive modules to manage, process and interpret the data. The user guide is organised by module.
