Principal interface and menu

 

The principal interface of the package bh_tomo is shown in fig. 2.1. The package is divided into interactive modules that allow one to create and manage a mini database, pick the travel times semi-automatically or automatically, process the travel time and amplitude data, adjust a covariance model to the data, perform the inversions, and treat time-lapse data. Both 2D and 3D models are available for the covariance, inversion and time-lapse modules. The buttons permit to execute different modules.
